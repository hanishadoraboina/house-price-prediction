# 🏡 House Price Prediction using Machine Learning

A Machine Learning project that predicts house prices based on various property features such as area, number of bedrooms, bathrooms, floors, location, condition, garage availability, and year built. This project demonstrates the complete machine learning workflow, including data preprocessing, exploratory data analysis (EDA), model training, evaluation, and prediction.

---

## 📌 Project Overview

Accurate house price prediction is valuable for buyers, sellers, and real estate professionals. This project leverages machine learning techniques to estimate property prices by analyzing multiple housing attributes.

The project covers every stage of the ML pipeline, from data cleaning to building predictive models, providing hands-on experience with real-world data analysis.

---

- Data loading and exploration

## 📂 Dataset

The dataset contains information about residential properties with features including:

- Property ID
- Area
- Bedrooms
- Bathrooms
- Floors
- Year Built
- Location
- Property Condition
- Garage Availability
- Price (Target Variable)
